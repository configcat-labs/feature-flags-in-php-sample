# feature-flags-in-php-sample
A companion repo for "How to use feature flags in PHP"


## How to run this app

1. Clone this repository

2. Run the following command from root:

```sh
composer install
```

3. Compile the frontend with NPM

```sh
npm run dev
```

4. Start the server:

```sh
php artisan serve
```
